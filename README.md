# W杯2026 全試合スケジュール（完全版）

FIFAワールドカップ2026（北中米共催）の **グループステージ72試合＋決勝トーナメント32試合＝全104試合** を、すべて日本時間で一覧できるWebアプリ。

## 機能
- 📅 **日程** — 日付・時間順の全104試合。放送局バッジ、本日ハイライト、LIVE/終了表示、国名検索、フィルタ（日本代表 / 注目 / 決勝T / 放送局別）
- ⏱ **カウントダウン** — 次の日本代表戦までをリアルタイム表示
- 🏆 **組分け** — A〜L組の順位表（試合終了後に自動反映）＋各組の日程
- 🗺️ **山（トーナメント表）** — ラウンド32〜決勝までの勝ち上がり表（横スクロール）
- 📖 **ルール** — 2022カタール大会との変更点（48チーム化、ラウンド32新設 ほか）
- 📲 **カレンダー登録** — 日本戦 / 表示中 / 全試合を `.ics` で書き出し（30分前アラーム付き）
- 📱 **PWA対応** — ホーム画面に追加してアプリのように起動、オフライン閲覧可

## 使い方
`index.html` をブラウザで開くだけ。サーバー不要。

## データ出典
FIFA公式・Wikipedia（2026 FIFA World Cup）。組み合わせ・放送スケジュールは確定情報に基づき更新されます。
